11.7 — Cartas Apiladas (Layered Offset Cards)

Composicion decorativa de tres capas para enmarcar elementos visuales protagonistas: videos, screenshots, mockups de producto. Dos rectangulos redondeados (--radius-lg) con rotaciones opuestas crean una ilusion de profundidad y anchoring espacial. La capa posterior usa Bio-Lime rotada –2.5°, la capa intermedia usa --color-bg-subtle rotada +2.5°, y el contenido principal flota en la capa frontal. Un dot grid ornamental (#CBD5E1) puntua una esquina opuesta. Uso reservado para el elemento visual dominante de secciones hero. No aplicar en elementos repetidos (tarjetas de blog, feature grids).

12 — Fotografia e Imagen

Estilo Visual

Las imagenes refuerzan la narrativa de tecnologia organica: data visualizations, abstractos con tonalidades verdes, e interfaces limpias en entornos profesionales.

Tratamiento

Fotografia con Overlay

Aplicar un overlay sutil de Bio-Lime (10-20% opacidad) sobre fotografias en blanco y negro para mantener coherencia con la paleta de marca.

Ilustraciones

Graficos de Datos

Preferir visualizaciones de datos abstractas con colores planos Bio-Lime sobre fondo oscuro. Estilo de network graph y nodos conectados. Nunca usar gradientes.

Texturas

Noise y Dot Grid

Usar texturas de ruido al 5% de opacidad y patrones de dot grid como fondos decorativos en lugar de imagenes literales. Los fondos son siempre colores planos.

Reglas de Imagen

Fondos abstractosColores planos + dot grid + noise. Nunca gradientes
Iconos e ilustracionesLineal, stroke 2px, estilo Lucide
13 — Modo Oscuro

Tema Oscuro

El modo oscuro es una parte integral de la identidad de Clicandseo. Los efectos bioluminiscentes cobran vida sobre fondos oscuros, creando una experiencia inmersiva.

Fondo

#0A0A0B

Fondo casi negro puro. Las tarjetas usan rgba(255,255,255,0.03) con bordes y sombras sutiles.

Texto

#F8FAFC

Blanco neutro para texto principal. Opacidades reducidas (60%, 40%) para texto secundario y terciario.

Efectos

Bio-Lime Glow

El Bio-Lime destaca especialmente sobre fondos oscuros. Usarlo como color de acento en bordes, textos destacados y elementos interactivos.

14 — Voz y Tono

Como Hablamos

La voz de Clicandseo es profesional pero accesible, tecnica pero comprensible. Hablamos con autoridad sobre SEO sin ser condescendientes.

Principio

Clara y Directa

Frases cortas. Sin jerga innecesaria. Si un termino tecnico es imprescindible, lo explicamos brevemente. Respetamos el tiempo del lector.

Principio

Data-Driven

Respaldamos afirmaciones con datos. Preferimos numeros concretos sobre adjetivos vagos. "Incrementa tu CTR un 40%" sobre "Mejora mucho tu CTR".

Principio

Empoderada

Posicionamos al usuario como protagonista. Las herramientas son el medio; el exito del usuario es el fin. "Tu estrategia, potenciada por IA".

Tono por Contexto

ContextoTonoEjemplo

Blog / ContenidoEducativo, autoridad"Analicemos como Google interpreta..."

Error / SoporteEmpatico, solucionador"No pudimos completar el analisis. Reintentando..."

15 — Uso Correcto

Hacer y No Hacer

Para mantener la coherencia de marca en todos los soportes, sigue estas directrices de uso correcto e incorrecto.

Logo

✓
Clicandseo.
Correcto

Usar el logo con los pesos y colores correctos. El punto siempre en Bio-Lime.

✕
clicandseo.
Incorrecto

No usar un peso uniforme. No escribir todo en minusculas sin diferenciacion tipografica.

Color

✓
Bio-Lime como acento
Correcto

Usar Bio-Lime como color de acento sobre fondos oscuros o claros. Nunca como color dominante de fondo.

✕
Texto blanco sobre lime
Incorrecto

No usar texto blanco sobre Bio-Lime (bajo contraste). No usar Bio-Lime como fondo completo de pagina.

✓
Lime sobre negro
Correcto

Bio-Lime siempre sobre fondo negro (#0A0A0B). Es la unica combinacion valida para textos e iconos en lime.

✕
Lime sobre blanco
Incorrecto

No usar Bio-Lime sobre fondo blanco. El contraste es insuficiente y el texto se vuelve ilegible.

✕
Lime sobre gris
Incorrecto

No usar Bio-Lime sobre fondo gris (#F1F5F9). Igual que sobre blanco, carece del contraste necesario para garantizar legibilidad.

Tipografia

✓
Inter Tight
con Libre Baskerville
Correcto

Combinar Inter Tight para titulares con Libre Baskerville italic para acentos editoriales.

✕
Arial Bold
con mas Arial
Incorrecto

No sustituir las fuentes de marca por fuentes genericas como Arial, Helvetica o system fonts.

Formas y Radios

✓
Correcto

Botones con radius-full (pill shape). Tarjetas con radius-md (20px). Formas suaves y organicas.

✕
Incorrecto

No usar esquinas rectas (border-radius: 0). Nuestra marca es "Ultra Soft" en todos los elementos.

Fondos y Efectos

✓
Correcto

Fondos siempre planos y solidos: #0A0A0B, #FFFFFF, #F1F5F9, #d9f9b8. Textura solo via noise al 5% o dot grid.

✕
linear-gradient
Incorrecto

Nunca jamas usar gradientes de ningun tipo: ni lineales, ni radiales, ni gradient meshes, ni degradados sutiles. Es una regla absoluta de la marca.

Efectos Prohibidos

✓
Sombras sutiles
shadow-sm, shadow-md, shadow-float
Correcto

Profundidad mediante sombras suaves (box-shadow). Colores solidos con opacidad controlada para capas.

✕
Glow, Glass, Orbs
backdrop-filter, blur, glow shadows
Incorrecto

No usar glow, orbes luminosos, glassmorphism, efectos translucidos ni blur decorativos. La marca es opaca, limpia y directa.
